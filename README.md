# interrupted_time_series_analysis.R

#### Interrupted Time Series Analysis

R script implementing Interrupted Time Series Analysis (ITSA) using segmented regression to detect structural changes in time-series data following an intervention.

Method applied in:
Ferreira et al. (2026) – Damming affects the taxonomic but not the functional beta diversity of fish in a neotropical reservoir ecosystem
